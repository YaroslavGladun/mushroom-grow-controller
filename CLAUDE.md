# CLAUDE.md

Цей файл описує контекст проєкту для майбутніх сесій Claude Code (та для людей).
Пиши й підтримуй документацію **українською**.

## Що це за проєкт

DIY-проєкт **автоматизованої клімат-камери для вирощування грибів** на базі
мікроконтролера **ESP32-WROOM-32 (DevKit V1)**. Камера — пластиковий контейнер
(60–80 л). Контролер автоматично регулює **температуру, вологість, освітлення**
та забезпечує **повітрообмін (FAE — Fresh Air Exchange)**.

Наразі репозиторій містить лише **документацію** (планування збірки та
закупівель). Прошивки/коду керування ще немає — це наступний етап.

## Зафіксовані умови (НЕ пропонувати альтернатив)

- **Контролер:** ESP32-WROOM-32 (DevKit V1). ~25 придатних GPIO (логіка 3.3 В),
  вбудовані Wi-Fi + Bluetooth, гнучкий I²C (за замовч. SDA=GPIO21 / SCL=GPIO22),
  16 апаратних ШІМ-каналів (LEDC), апаратний watchdog. Без ОС — прошивка на
  **ESPHome** (рекоменд.) / Arduino (C++) / MicroPython. Для навантажень уникати
  strapping-пінів (0, 2, 5, 12, 15), UART0 (1/3), пінів SPI-flash (6–11) та
  input-only (34/35/36/39).
- **Камера:** пластиковий контейнер 60–80 л, висока внутрішня вологість.
- **Обов'язкова автоматика:** температура, вологість, світло + повітрообмін.

## Цільові параметри середовища

| Фаза          | Температура | Вологість (RH) | Світло       |
| ------------- | ----------- | -------------- | ------------ |
| Колонізація   | 24–27 °C    | 80–90 %        | без світла   |
| Плодоношення  | 21–24 °C    | 90–95 %        | цикл 12/12   |

## Ключові інженерні орієнтири (для узгодженості документів)

Ці рішення вже обґрунтовані в `docs/` — тримай їх узгодженими між файлами:

- **Датчик:** I²C-датчик t°/RH, стійкий до конденсату (пріоритет — SHT-серія).
- **Комутація навантажень:** SSR (DC-вхід) для мережевого 220 В нагрівача;
  логічні MOSFET-модулі (D4184/IRLZ44N) для DC/ШІМ (туманоутворювач 24 В, LED,
  вентилятори). Тригер 3.3 В підходить напряму. Керуючі піни — LOW на старті:
  зовнішній 10 кОм pull-down **обов'язковий** (до запуску прошивки піни ESP32 у
  high-Z), для навантажень брати лише «безпечні» GPIO, не strapping-піни.
- **Живлення:** три ізольовані рейки — 5 В (ESP32, у VIN/USB) + 12 В (LED/вентилятори)
  + 24 В (туманоутворювач); спільна «земля» в одній зірковій точці.
- **Захист електроніки:** ESP32, реле та БЖ — ЗЗОВНІ контейнера в герметичному боксі
  (IP65+), всередину заходять лише датчик і трубки. Гермовводи (cable glands). Бокс
  ABS/пластик радіопрозорий — Wi-Fi ESP32 працює зсередини боксу.
- **Стерильність:** ізопропіл/спирт 70 %, HEPA-фільтр на притоці FAE, мікропористий
  скотч, рукавиці; SAB (still-air-box) для роботи зі спавном.
